# Minimalist Tic-Tac-Toe AI (CLI)

A clean, entirely terminal-based implementation of Tic-Tac-Toe, powered by mathematical optimization. 

Developed for the **CSA2001 Bring Your Own Project (BYOP)** component, this application leverages the **Minimax algorithm** in Python. The AI opponent evaluates the entire tree of possible future game states, rendering it mathematically incapable of losing. The project strictly adheres to a Command Line Interface (CLI) execution model with zero GUI dependencies.

---

## ⚙️ Prerequisites & Dependencies

This project is built using standard Python libraries to maintain a lightweight footprint. 
* **Requirement:** Python 3.6 or higher installed on your system.
* **Dependencies:** No external pip packages or virtual environments are required.
